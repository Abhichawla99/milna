# 🚀 How to Embed Floral AI Widget on Any Website

## 📋 Simple Instructions for Website Owners

### 🎯 **Option 1: Complete Widget (Recommended)**
**Best for: Users who want everything to work immediately**

1. **Copy this code:**
   ```html
   <script src="https://yourdomain.com/floral-ai-complete-widget.js"></script>
   ```

2. **Paste it into your website** before the `</body>` tag

3. **Done!** Your AI chat widget is now live with full backend integration

**✅ Includes:** Backend integration, user tracking, message limits, AI responses, analytics

---

### 🎯 **Option 2: Basic Widget**
**Best for: Developers who want to customize the backend integration**

1. **Copy this code:**
   ```html
   <script src="https://yourdomain.com/embed-agent.js"></script>
   ```

2. **Paste it into your website** before the `</body>` tag

3. **Configure** your backend endpoints as needed

**⚙️ Requires:** Manual backend setup, webhook configuration

---

## 🌐 **Where to Add the Code**

### WordPress
1. Go to **Appearance > Theme Editor**
2. Select **footer.php** 
3. Add the script before `</body>`
4. Click **Update File**

### Shopify
1. Go to **Online Store > Themes**
2. Click **Actions > Edit Code**
3. Open **layout/theme.liquid**
4. Add the script before `</body>`
5. Click **Save**

### HTML Website
1. Open your HTML file
2. Find the `</body>` tag
3. Add the script right before it
4. Save the file

### Squarespace
1. Go to **Settings > Advanced > Code Injection**
2. Paste the script in **Footer**
3. Click **Save**

### Wix
1. Go to **Settings > Custom Code**
2. Click **Add Custom Code**
3. Paste the script and set to "Body - end"
4. Click **Apply**

## 🚨 **Troubleshooting**

### Widget Not Appearing?
- Check if the script is before `</body>`
- Verify the script URL is correct
- Check browser console for errors

### Not Getting Responses?
- Ensure you're using the complete widget version
- Check your internet connection
- Wait a few seconds for AI processing

### Styling Issues?
- The widget uses high z-index (999999)
- Check for CSS conflicts
- Try the different position options
